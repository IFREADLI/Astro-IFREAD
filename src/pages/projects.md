## 个人项目

### 博客系统

- **技术栈**：Astro、TypeScript、Tailwind CSS
- **描述**：使用Astro构建的个人博客系统，支持Markdown文章、标签分类、搜索等功能
- **特点**：
  - 静态生成，加载速度快
  - 响应式设计，支持移动端
  - 深色模式切换
  - SEO优化
- **链接**：[GitHub仓库](https://github.com/lijnf/blog)

### 任务管理应用

- **技术栈**：React、Node.js、MongoDB
- **描述**：一个简单的任务管理应用，支持任务创建、编辑、删除和状态管理
- **特点**：
  - 拖拽排序功能
  - 实时同步
  - 数据可视化
  - 移动端适配
- **链接**：[在线演示](https://tasks.lijnf.top)

### 天气应用

- **技术栈**：Vue、Vite、OpenWeather API
- **描述**：一个简洁的天气应用，显示当前天气和未来几天的预报
- **特点**：
  - 地理位置定位
  - 多城市管理
  - 天气数据可视化
  - PWA支持
- **链接**：[GitHub仓库](https://github.com/lijnf/weather-app)
